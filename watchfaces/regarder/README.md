# Regarder Watchface
Watchface for Amazfit GTR Mini and for other round ZeppOS watch.

![demo](./demo-gtr-mini.png)

## Description
 Customizable minimalist watchface. Features multiple background (bezel) styles, 4 complications (15+ data types including sleep duration, weather, sunrise/sunset, biocharge, alarm, world time), and AOD. Inspired by the Regarder face for Android Wear.

**Original idea by:**
Watchface [Regarder Minimal 2](https://play.google.com/store/apps/details?id=com.watchfacestudio.regarder_minimal_2) by Regarder for Android Wear watches.
