# atan2 Watchface
Watchface for Amazfit GTR Mini and for other round ZeppOS watch.

![demo](./demo-gtr-mini.png)

## Description
Animated watch face with hour and minute hands formed by a polygon mesh of graph edges. The central elements rotate every second. Includes date (always in negative space), battery indicator, and AOD.

**Original idea by:**  
[Watchface by AlanSmithee for Garmin watches](https://apps.garmin.com/apps/7c6a06f6-1582-46b3-9923-9ae9dd82418f)
