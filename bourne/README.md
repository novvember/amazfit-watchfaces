# Bourne Watchface
Watchface for Amazfit GTR Mini

![demo](./demo.png)

## Features

**Main features**
- inspired by [TTMMBRN watchface](https://ttmm.is/portfolio/ttmmbrn/)
- use font from Jason Bourne movie
- heart rate
- time
- date
- temperature
- steps

**Model compatibility:** Amazfit GTR Mini (round, 416 x 416 pixels)

**AOD:** Yes

**Tap-zones:** No

**Language:** English
