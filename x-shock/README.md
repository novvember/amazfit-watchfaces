# X-Shock Watchface
Watchface for Amazfit GTR Mini and for other round ZeppOS watch.

![demo](./demo-gtr-mini.png)

## Description

Lots of information including sun and moon info, PAI (total and for latest 6 days), and different system statuses. AOD (only time) as well. Inspired by different popular G-Shock models.

**Language:** English

**Reference:** Casio G-Shock watches design
