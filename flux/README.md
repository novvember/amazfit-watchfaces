# Flux Watchface
Watchface for Amazfit GTR Mini

![demo](./demo-gtr-mini.png)

## Features

Copy of the famous Apple Watch Flux watchface.

**Main features**
- Only current time with digits showing hours and minutes.
- The horizontal line (level) shows current second value.
- Only one color theme yet.
- Only one digits configuration yet.

**Reference:** [Apple Watch Flux watchface](https://support.apple.com/guide/watch/faces-and-features-apde9218b440/watchos)

**Model compatibility:** All round watches

**AOD:** No

**Tap-zones:** No

**Language:** English / Russian
