# Particles Watchface
Watchface for Amazfit GTR Mini

![demo](./demo.png)

## Features

**Main features**
- background lines move towards current seconds position and also change color and thickness
- time in the center
- date and weekday
- steps

**Model compatibility:** Amazfit GTR Mini (round, 416 x 416 pixels)

**AOD:** No

**Tap-zones:** No

**Language:** English
